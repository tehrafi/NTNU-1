Questions

  1. What does the server in the client server architecture do?
    The server holds the resources the client want. When it gets a request from the client, it delivers the right resource.
  2. What does the client do?
    The client "asks" the server for resources and gets them returned.
  3. What is an IP address?
    "IP" stands for Internet protocol. And IP address is numerical, unique label assigned to each computer/device connected to the internet.
    There are two different versions of an IP, IPv4 and IPv6. IPv4 holds only numbers,
    IPv6 holds both numbers and characters. It is used for identification and for for example a client to connect to a server.
  4. What is DNS?
    DNS stands for Domain Name System. It is used for "better internet experience". A DNS replaces a webpage's IP with text,
    so humans can better remember and more easily enter webpages.
    For example: Goolge's DNS is "2a00:1450:400f:804::200e" (IPv6), but thei'r DNS is "www.Google.com".
  5. What is the difference between the Internet and WWW?
    The Internet is a massive network of networks. It connects devices together, so every device can communicate with any other.
    The WWW is a way of accessing information through the Internet.
  6. Explain what HTML is
    HTML stands for "Hyper Text Markup Language". It is a programming language used for building websites.
    It can be written in any text editor. The browser reads the HTML and shows it at a website.
